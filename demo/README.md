# vida Demo Deployment

A browser-based demo of the vida desktop app. Uses the real UI with generated fixture data and a React Three Fiber virtual live feed.

## Local Development

```bash
cd demo
npm install
npm run dev
```

## Deploying to Vercel

1. Import the repository into Vercel.
2. Set the project root to `demo/`.
3. Use the default Vite build command: `npm run build`.
4. Use the output directory generated by Vite (`dist/`).
5. Point the repository About URL and README badge to the deployed demo.

## What the Demo Includes

- Full vida timeline, dashboard, search, and detail panel UI
- Generated fixture data (frames, summaries, events, activities)
- Schedule-driven simulation that produces realistic activity patterns
- React Three Fiber 3D scene as a virtual live feed (replaces MJPEG camera stream)
- Read-only mode: Settings, onboarding, and memos are view-only with "Demo" labels
